# FreyPlatonic

An infinite platformer where you need to destroy all your enemies!

![Frey Platonic](./assets/img/screenshot-1.jpg)


## Try it out!

https://spiffy-swan-c9ff23.netlify.app/

## Controls

1. `WASD` movement
2. `Space` jumping
3. `Left mouse` attack
4. `Right mouse` block
5. `1-4` select items
6. `E` use item
